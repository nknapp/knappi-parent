This site contains information about some of my open-source projects.

External-Tools

Aims to be a set of java wrappers to invoke external tools that are useful in everyday life e.g.
  • kill, hangup processes
  • send notifications to the user (via notify-send)
  • ...
Currently only support for linux.

Calendar-Widgets

Aims to be a extensible Swing widget the display and modification of time-table like data.
